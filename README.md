# 📚 JavaLibrary_20

A Library Management System developed in Java as part of **Software Design 150s coursework at CPUT**.  
This project demonstrates teamwork, software design principles, and practical implementation using Java Frame Forms.

## 🧠 Project Overview
This system automates library operations such as book registration, borrowing, and return tracking.  
It integrates constructors, modifiers, control statements, and GUI forms to ensure usability and maintainability.
